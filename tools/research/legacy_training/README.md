# 旧策略训练工具集

本目录是主项目早期遗留的简单训练脚本，只用于追溯历史参数来源。

当前正式训练与知识库不在主项目维护；主项目只保留部署、实盘运行、观察标签和诊断工具。

## 文件说明

| 文件                | 用途                                      |
| :------------------ | :---------------------------------------- |
| `train_strategy.py` | 10轮滚动窗口训练 - 昨日涨停续涨准确率验证 |

## 使用方法

```bash
cd tools/research/legacy_training
python train_strategy.py
```

训练结果会输出各组合条件（高开区间 x 缩量比）的验证准确率，并写入本目录的 `training_results.json`。

## 已有训练成果

| 条件                 | 样本 | 验证准确率 |
| :------------------- | :--- | :--------- |
| 高开 ≥5% + 缩量 <0.8 | 115  | **63%**    |
| 高开 ≥3%             | 398  | 41%        |
| 高开 ≥7% + 缩量 <0.8 | 85   | 49%        |

> 结论：高开 ≥5% + 缩量 <0.8 是最佳参数组合，已合并进 `auction.zt_continue_boost` 配置。
