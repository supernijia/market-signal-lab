# 个股深度体检分析手册 (Analyze Mode - V5.0)

本手册介绍如何使用 `analyze` 模式对单只个股进行 360 度多维"深度体检"。V5 版本在 V4 的 T+0 可视化基础上，全面引入了 **多周期共振扫描、高级技术指标诊断（KDJ/BOLL/W&R/OBV）、10 日累积资金流追踪、大盘 Alpha 对标** 以及 **持仓风控量化（最大回撤/波动率）**。

---

## 🚀 核心功能概述

| 模块           | 功能                                                     | 数据来源                                 |
| -------------- | -------------------------------------------------------- | ---------------------------------------- |
| **实时行情**   | 价格/涨跌幅/高低点/全天成交额                              | Tushare `rt_k` |
| **技术指标**   | MA5/10/20/30/60/120, MACD, RSI, ATR, KDJ, BOLL, W&R, OBV | `TechAnalyzer`                           |
| **多周期共振** | 日/周/月 趋势判断 + MACD/KDJ/BOLL/WR 共振信号评分        | `TechAnalyzer`                           |
| **基本面雷达** | PE(TTM)/PB/ROE/负债率 + 财务排雷                         | Tushare `daily_basic` + `fina_indicator` |
| **资金流追踪** | 10 日累积主力净流入                                      | Tushare `moneyflow_dc`（`moneyflow` 兜底） |
| **大盘对标**   | 个股 vs 上证指数 20 日 Alpha 计算                        | Tushare `index_daily`                    |
| **题材透视**   | 热门概念板块关联                                         | Tushare `concept_detail`                 |
| **机构调研**   | 近 30 天调研次数与机构数                                 | Tushare `stk_surv`                       |
| **持仓风控**   | 区间最大回撤 / 年化波动率 (需传 `--buy-price`)           | `utils.py`                               |
| **T+0 可视化** | 日线图 + 分时图 + VWAP + 网格做 T 建议                   | `ChartEngine`                            |

---

## 🛠️ 如何运行

```bash
# 基础诊断
python main.py --mode analyze --code 000807

# 深度 T+0 复盘 (带买入成本、时间与持仓信息)
python main.py --mode analyze --code 000807 --buy-price 34.50 --buy-time "2026/03/05 10:00" --hold-vol 1000
```

### 参数说明

| 参数          | 说明                                         | 示例                 |
| ------------- | -------------------------------------------- | -------------------- |
| `--code`      | 股票代码（支持 `000807.SZ` 或简写 `000807`） | `000807`             |
| `--buy-price` | 持仓底仓成本价，启用持仓风控与图表成本线     | `34.50`              |
| `--buy-time`  | 持仓建立的时间，用于图中垂直标记             | `"2026/03/05 10:00"` |
| `--hold-vol`  | 持仓股数                                     | `1000`               |
| `--no-email`  | 仅在终端输出报告，不发送邮件                 | -                    |

## 📊 V5 报告结构

报告正文依次包含以下章节：

### 一、核心概况与行情

- 实时价格/换手/成交额/技术状态/上市板块
- **大盘对标**: 个股 20 日涨幅 vs 上证指数 20 日涨幅 → Alpha 差值
- **主力动向**: 10 日累积主力净流入/净流出

### 二、持仓风控与实战考量 _(需传 `--buy-price`)_

- 建仓成本 → 当前浮盈
- **区间最大回撤** / **年化波动率** (量化持仓期间真实风险敞口)
- 基于止损距离的操作建议 (自动识别危险/安全/突破状态)

### 三、价值底与资金热度 (V3 专版)

- **财务核心指标**: PE(TTM)/PB/ROE/负债率 _(PE/PB 来自 `daily_basic` 接口)_
- 核心题材标签、机构调研、融资融券杠杆资金

### 四、多指标多周期共振扫描

- **跨周期趋势表**: 日线(MA20)/周线(MA60)/月线(MA120) 红绿灯
- **指标共振度评分**: MACD/KDJ/BOLL/W&R 四大指标的方向性信号统计 (`/4`)
- ATR 日均波幅、支撑位、阻力位、量化止损点

### 五、交易系统策略执行建议

- 当前状态判定 (强势多头/震荡偏强/震荡偏弱/极度弱势)
- 动能强度描述

### 六、T+0 分时与日线全景导引图 (V4 专版)

- 日线图 (含持仓成本线)
- 分时图 (含做 T 阻力支撑与 VWAP 均线)
- 日内网格做 T 指导策略 (Pivot/R1/R2/S1/S2)

---

## 📉 图表解析

### 1. 日线趋势图 (Daily Chart)

- **数据源**: Tushare `pro.daily`
- **标记线**: 过去 60 个交易日 K 线走势，红色水平虚线为 **`buy-price`** 成本位。

### 2. 日内分时图 (Intraday Chart)

- **双接口保障**: 优先 Tushare 1min 数据；额度耗尽时自动切换至 **腾讯财经 (GTimg)** 接口（可通过 `.env.openclaw` 的 `TENCENT_MINUTE_FALLBACK_*` 环境变量配置或禁用）。
- **做 T 锚点**: 蓝色水平线为成本位，蓝色垂直线为 **`buy-time`**。
- **黄线 (VWAP)**: 当日成交量加权平均价线。
- **未来预测区间**: 时轴自动外延至 **15:00 收盘**，右侧空白投影支撑与阻力线。

---

## 💡 网格做 T 策略解读

- **多空枢轴 (Pivot)**: 全天强弱的分水岭。
- **超买抛压区 (R1/R2)**: 股价上冲至此建议分批"高抛"。
- **超卖低吸区 (S1/S2)**: 股价回落至此建议"低吸"回补。
- **量化止损点**: 基于 ATR 动态波幅的铁律止损线，跌破无条件离场。
